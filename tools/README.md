# Tools

Pomocnicze skrypty deweloperskie. Nie są częścią głównej aplikacji.

## force_read_reports.py
Wymusza ponowne odczytanie raportów z cache. Używaj gdy raporty są uszkodzone lub niekompletne.

## raw_troops.py
Narzędzie do debugowania parsowania danych o wojskach.

## Użycie
```bash
python tools/force_read_reports.py
python tools/raw_troops.py
```
